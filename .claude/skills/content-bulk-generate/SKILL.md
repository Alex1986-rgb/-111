# Content Bulk Generate - Массовая генерация контента

Автоматическая генерация описаний товаров, meta-тегов, FAQ с умным распределением между Haiku (простые) и Sonnet (экспертные).

## Использование

```bash
/content-bulk-generate <input-file> [options]
```

## Опции

- `--type <simple|expert>` - Тип контента (по умолчанию: auto)
- `--count <number>` - Количество элементов для генерации
- `--export <format>` - Формат экспорта: csv, json, xlsx (по умолчанию: csv)
- `--tone <voice>` - Tone of voice: professional, casual, luxury, technical
- `--region <name>` - Региональная уникализация (Москва, СПб, Екатеринбург)
- `--batch-size <number>` - Размер батча (по умолчанию: 100)

## Примеры

```bash
# Простая генерация из CSV
/content-bulk-generate products.csv --type simple --count 1000

# Экспертный контент с региональной уникализацией
/content-bulk-generate products.csv --type expert --region "Москва" --tone professional

# Автоматический выбор модели
/content-bulk-generate products.csv --export json,xlsx
```

## Автоматическое распределение моделей

### Haiku 4.5 (быстро и дешево)
**Используется для:**
- Простые описания товаров по шаблону
- Meta-теги (Title, Description)
- Alt-теги для изображений
- Короткие FAQ
- Характеристики

**Стоимость:** ~$0.00025 за 1000 токенов
**Скорость:** ~1000 товаров за 5 минут

### Sonnet 4.5 (качественно)
**Используется для:**
- Экспертные статьи
- Уникальный контент
- Сложные описания
- PR-материалы
- Технические тексты

**Стоимость:** ~$0.003 за 1000 токенов
**Скорость:** ~100 товаров за 5 минут

## Шаблоны контента

### Простой товар (Haiku)
```
Title: {name} | Купить в {region}
Meta: {name} - {category}. Цена от {price}₽. Доставка по {region}.
Description: {name} от {brand}. {characteristics}. Заказывайте онлайн!
```

### Экспертный товар (Sonnet)
```
Title: {name} - полный обзор и характеристики | {brand}
Meta: Подробный обзор {name}: характеристики, отзывы, цены. Сравнение с аналогами.
Description: Глубокий анализ с экспертными рекомендациями, реальными отзывами...
```

## Региональная уникализация

Автоматическая адаптация под регион:
- Москва: "Доставка за 2 часа", "Самовывоз из 50 точек"
- СПб: "Доставка по Санкт-Петербургу", "Пункты выдачи в центре"
- Регионы: "Доставка по России", "Оплата при получении"

## Интеграция с CMS

```php

  --post_title="Смартфон X Pro" \
  --meta_input='{"_yoast_wpseo_title":"..."}' \
  --from-csv=products.csv
```

```php

$product->Add([
  'IBLOCK_ID' => 1,

  'DETAIL_TEXT' => $data['description']
]);

```
