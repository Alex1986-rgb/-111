# TextFlow — Экосистема экспертного копирайтинга и SEO-редакции

![TextFlow Hero](https://images.unsplash.com/photo-1519389950473-47ba0277781c?auto=format&fit=crop&q=80&w=1200)

**TextFlow** — это высокотехнологичное решение для агентств копирайтинга, ориентированное на российский рынок (РФ). Платформа объединяет в себе продающий лендинг, интеллектуальный калькулятор услуг, систему экспертного SEO-аудита и полноценную панель управления заказами с интегрированным платежным шлюзом.

---

## 🌟 Основные преимущества

*   **Редакционный ассистент**: Помогает разрабатывать идеи и структуру статей прямо в интерфейсе.
*   **Прозрачное ценообразование**: Калькулятор в реальном времени рассчитывает стоимость на основе объема (в 1000 знаков) и типа услуги.
*   **LSI-Оптимизация**: Все тексты и структура сайта заточены под современные требования поисковых систем (Яндекс/Google).
*   **Безопасные сделки**: Встроенный интерфейс оплаты с поддержкой СБП и банковских карт.

---

## 🚀 Ключевые особенности

- **🔍 Экспертный аудит текста**: Инструмент для мгновенной проверки контента на SEO-оптимизацию, "воду" и читаемость по стандартам редакции.
- **📦 Массовые заказы (Bulk Orders)**: Система автоматизированного формирования ТЗ для сотен карточек товаров с поддержкой импорта из Excel/CSV.
- **💳 Платежный шлюз (Эквайринг)**: Имитация оплаты через СБП и Карты РФ с защитой транзакций.
- **📊 Admin Dashboard**: Профессиональная панель управления ценами, статусами заказов и ответами клиентам.
- **📱 Ultra-Responsive Design**: Идеальное отображение на любых устройствах: от смартфонов до 4K мониторов.
- **🧠 Интеллектуальный бриф**: Двухэтапная форма заказа с выбором Tone of Voice и целевой аудитории.
- **📈 SEO-ready**: Встроенная микроразметка Schema.org, мета-теги и оптимизированная структура для продвижения в Яндекс.
- **📚 Экспертный блог**: 16+ глубоких статей с аналитикой и трендами копирайтинга 2025 года.

## 🛠 Технологический стек

- **Frontend**: React 19, TypeScript
- **Styling**: Tailwind CSS 4.0 (Modern Glassmorphism)
- **Icons**: Lucide React
- **AI Engine**: Google GenAI (Gemini 3 Flash Preview)
- **State Management**: React Context API + Hooks + LocalStorage Persistence
- **Build Tool**: Vite

## 📦 Установка и запуск

1. **Клонирование репозитория**
   ```bash
   git clone https://github.com/your-username/textflow.git
   cd textflow
   ```

2. **Установка зависимостей**
   ```bash
   npm install
   ```

3. **Настройка окружения**
   Для работы AI-функций необходимо наличие ключа `GEMINI_API_KEY` в системном окружении или файле `.env`.

4. **Запуск проекта**
   ```bash
   npm run dev
   ```

## 📖 Инструкция по использованию

### 👤 Для клиента
1. **Расчет стоимости**: Используйте калькулятор на главной странице или пройдите квиз для подбора тарифа.
2. **Аудит**: Вставьте свой текст в блок "Экспертный аудит" для получения мгновенных рекомендаций по улучшению.
3. **Оформление**: Заполните бриф (Email, описание, стиль текста).
4. **Оплата**: Выберите удобный способ (Карта или СБП) и подтвердите транзакцию.

### 🔑 Для администратора
1. **Доступ**: Перейдите в футер и нажмите "Панель администратора".
2. **Управление ценами**: Вкладка "Цены" позволяет менять тариф за 1000 знаков для каждой услуги в реальном времени.
3. **Обработка заказов**: 
   - Меняйте статусы (В работе / Завершено).
   - Проверяйте статус оплаты заказа.
   - Отправляйте клиенту ответ (например, ссылку на Google Doc с готовым текстом) через поле "Ответ администратора".

## 📂 Структура проекта

- `/src/components`: Набор переиспользуемых UI-компонентов (Hero, Admin, Payment и др.).
- `/src/types.ts`: Строгая типизация данных (Заказы, Услуги, Статьи).
- `/src/constants.ts`: Конфигурационные данные, тексты статей и начальные настройки цен.
- `/src/App.tsx`: Корневой компонент с логикой маршрутизации и управления состоянием.
- `/src/index.css`: Глобальные стили и конфигурация Tailwind.
